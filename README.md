# designerweb.cz landing page

Modern, high-converting one-page landing page for a web design agency.

## Run

- **Fastest**: open `index.html` in your browser.
- **Recommended** (best for smooth scrolling + relative assets):

```bash
# If you have Node.js:
npx serve .
```

Then open the local URL it prints (usually `http://localhost:3000`).

## Customize

- **Email**: `index.html` + `main.js` uses `hello@designerweb.cz`
- **Copy**: edit sections directly in `index.html`
- **Brand colors**: tweak `--accent-a/b/c` in `styles.css`
